# Wikibase Everywhere

Welcome to the Wikibase Everywhere project! This project aims to provide a simple way to interact with a local wikibase instance.

## Installation

Create a.env file containing:

```bash
DB_NAME=wikibase
DB_USER=wikibase
DB_PASS=wikibase
```

Then run the following commands:

```bash
docker-compose up -d
```
